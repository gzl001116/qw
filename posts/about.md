# 关于这个博客

date: 2023-01-02

## 技术栈

这个博客使用了以下技术：

- Node.js - 运行环境
- Express.js - Web框架
- EJS - 模板引擎
- Marked - Markdown解析
- Highlight.js - 代码高亮

## 部署方式

博客可以轻松部署到各种平台，包括：

1. GitHub Pages
2. Heroku
3. Vercel
4. 云服务器

## 如何使用

1. 克隆项目
2. 安装依赖: `npm install`
3. 创建文章: 在 `posts` 目录下创建 Markdown 文件
4. 启动服务: `npm start`

## 自定义

你可以轻松地自定义这个博客：

- 修改模板文件 (`views` 目录)
- 调整样式 (`public/css` 目录)
- 添加新功能到 `index.js`

欢迎提交 Pull Request 来改进这个项目！
